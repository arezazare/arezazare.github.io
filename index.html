About Me

IT Support Analyst → Systems Automation | PowerShell • Bash • Python • Batch | Active Directory • Microsoft 365 | Endpoint Operations

I am A. Reza Zare, an IT Infrastructure & Cloud Engineer and Automation Specialist with a Master’s degree in Computer Science and 8+ years of experience spanning IT operations, internal tools development, and automation.

I started my career in IT support, providing end-user assistance and managing enterprise systems, and gradually moved toward systems automation, identifying repetitive operational tasks and replacing them with efficient scripting solutions.

At YM Inc., I support a 1000+ user environment across corporate, retail, and warehouse operations. I develop automation tools using PowerShell, Python, Bash, and Batch for tasks such as Active Directory auditing, endpoint cleanup, operational reporting, and workflow optimization. I also manage account and endpoint operations in Microsoft 365, configure MFA (Duo), VPN (GlobalProtect), monitor security via SentinelOne, and handle enterprise print infrastructure.

Previously, at Medrick, I spent five years designing and maintaining internal tools, developing automation utilities, and building SQL/PHP scripts to support operational reporting, system monitoring, and workflow improvements.

My technical focus spans:

  • Cloud Infrastructure: Windows Server, Active Directory, GPO, DNS/DHCP, virtualization (VMware, Hyper-V, VirtualBox), Azure, and Entra ID.

  • Scripting: PowerShell, Python, Bash, Batch, SQL — for reporting, auditing, and workflow automation.

  • Security Management: Endpoint monitoring, compliance, MFA, VPN, Intune, ManageEngine Endpoint Central.

  • Data Analytics: Web applications, dashboards, APIs, and data analytics with Python, SQL, Tableau, Power BI, HTML, CSS, JavaScript, PHP/Laravel.
