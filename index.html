About Me

IT Support Specialist | Automation Developer | PowerShell • Python | Active Directory • Microsoft 365 • Endpoint Operations | Full-stack Developer

I am A. Reza Zare, an IT Support Specialist and Automation Developer with a Master's degree in Computer Science and 8+ years of experience spanning IT operations, internal tools development, and scripting automation.

I started my career in IT support and gradually moved toward systems automation — identifying repetitive operational tasks and replacing them with efficient scripting solutions.

At YM Inc., I support a 1,000+ user environment across corporate, retail, and warehouse operations. I build automation tools using PowerShell and Batch for tasks such as Active Directory auditing, endpoint cleanup, disk maintenance, and system health validation — deployed fleet-wide via ManageEngine Endpoint Central. I also support Active Directory, Microsoft 365, endpoint security via SentinelOne, and enterprise print infrastructure.

Previously at Medrick, a game publisher in the MENA region, I spent five years building internal web tools and automation utilities using PHP/Laravel, JavaScript, Python, and SQL — supporting operational reporting, system monitoring, and workflow improvements for the ops and support teams.

I also independently built a complete e-commerce web application using PHP/Laravel, HTML/CSS/SASS, JavaScript, and MySQL — handling everything from the database layer to the front-end interface.

My technical focus spans:

  • IT Infrastructure: Windows Server, Active Directory, GPO, DNS/DHCP, Azure/Entra ID, endpoint management.

  • Scripting & Automation: PowerShell, Python, Bash, Batch, SQL — auditing, reporting, and workflow automation.

  • Web Development: PHP/Laravel, HTML/CSS/SASS, JavaScript, MySQL — full-stack internal tools and web applications.

  • Development & Programming: Python, C# / .NET, Git/GitHub — scripting, automation, and application development.
